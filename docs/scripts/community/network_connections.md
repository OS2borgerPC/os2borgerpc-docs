## Beskrivelse
Viser BorgerPC'ens eller Kiosk-maskinens netværksenheder og -adresser.
Informationen kan findes i job-loggen, efter at jobbet er blevet udført.

Dette script er blevet testet og virker på Ubuntu 22.04.
